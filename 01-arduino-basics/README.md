# 01 Arduino Basics

Fundamentals of microcontroller programming: GPIO, digital/analog I/O, and basic sensors.

## Exercises

| # | Exercise | Concepts |
|---|----------|----------|
| 1 | blink | Digital output, delay() |
| 2 | external-led | GPIO pins, breadboard wiring |
| 3 | fade | PWM, analogWrite() |
| 4 | button | Digital input, pull-up resistors |
| 5 | button-toggle | State machines |
| 6 | relay | High-power switching |
| 7 | potentiometer | Analog input, map() |
| 8 | tof-sensor | I2C, VL53L0X library |
| 9 | tof-led | Sensor-actuator integration |
| 10 | integration | Combined project |

## Hardware

- ESP32 DevKit
- Breadboard + jumper wires
- LEDs (red, green, blue)
- 220 ohm resistors (for LEDs)
- 10k ohm resistors (for buttons)
- Push buttons
- Potentiometer (10k)
- VL53L0X ToF sensor
- Relay module (5V)
