# ESP32 Fundamentals

Open-source tutorials for microcontroller programming with ESP32 development boards.

## Modules

1. **[01-arduino-basics](01-arduino-basics/)** - GPIO, LEDs, buttons, sensors
2. **02-esp-now** - Peer-to-peer communication (coming soon)
3. **03-mqtt** - Networked messaging (coming soon)
4. **04-apis** - Web services (coming soon)
5. **05-osc** - Creative applications (coming soon)

## Prerequisites

- ESP32 DevKit or similar board
- Arduino IDE with ESP32 board support
- Basic electronics: breadboard, LEDs, resistors, buttons

## Getting Started

1. Install [Arduino IDE](https://www.arduino.cc/en/software)
2. Add ESP32 board support: File > Preferences > Additional Board URLs:
   ```
   https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
   ```
3. Install ESP32 boards: Tools > Board > Boards Manager > Search "esp32"
4. Select your board: Tools > Board > ESP32 Arduino > ESP32 Dev Module
