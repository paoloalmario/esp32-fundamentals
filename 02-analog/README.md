Analog input reading on ESP32.

## Exercises

| Exercise | Concepts | Hardware |
|----------|----------|----------|
| [potentiometer](potentiometer/) | ADC, analog read | Potentiometer or LDR |

## Key Concepts

- **ADC**: Analog-to-Digital Converter
- **12-bit resolution**: Values 0-4095
- **Reference voltage**: 3.3V
- **ADC pins**: GPIO 32-39 recommended (ADC1)
